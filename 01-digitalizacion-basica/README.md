# Digitalización básica

Este bloque es tu punto de partida si nunca has programado.

**Objetivo:** Reducir vértigo, generar confianza y entender el terreno antes de profundizar.

---

## Sobre este bloque

Este NO es un bloque técnico profundo. Es un aterrizaje suave.

Aquí vas a:
- Entender qué es un ordenador, internet y el desarrollo web sin tecnicismos
- Ver con tus propios ojos qué hace el frontend y qué hace el backend
- Escribir tu primera línea de código y ver resultados en pantalla
- Tocar, romper y arreglar cosas en un entorno seguro
- Comprender que experimentar es parte del proceso, no un error

**Después de este bloque, irás a [/02-frontend/fundamentos](../02-frontend/fundamentos/README.md) para construir base técnica sólida.**

---

## Los módulos

### `01-que-es-un-ordenador.md`
**Descripción:** Cómo piensa una máquina, qué es el hardware, el software y el sistema operativo.  
**Nivel:** Esencial

### `02-que-es-internet.md`
**Descripción:** Cómo funciona internet por encima. Qué pasa cuando visitas una web.  
**Nivel:** Esencial

### `03-que-es-desarrollo-web.md`
**Descripción:** Qué hace un desarrollador web. Diferencia entre crear contenido y programar.  
**Nivel:** Esencial

### `04-frontend-backend-visual.md`
**Descripción:** Qué ves (frontend) vs qué no ves (backend). Ejemplos visuales con webs conocidas.  
**Nivel:** Esencial

### `05-herramientas-basicas.md`
**Descripción:** Navegador, editor de código, consola del navegador. Para qué sirve cada uno.  
**Nivel:** Esencial

### `06-primer-documento-html.md`
**Descripción:** Crear tu primer archivo HTML. Ver algo en pantalla. Sin teoría profunda todavía.  
**Nivel:** Esencial

### `07-primer-toque-css.md`
**Descripción:** Cambiar colores y tamaños. Ver que CSS modifica la apariencia.  
**Nivel:** Esencial

### `08-primer-toque-javascript.md`
**Descripción:** Hacer que algo cambie al hacer clic. Ver que JavaScript hace cosas dinámicas.  
**Nivel:** Esencial

### `09-que-es-dom.md`
**Descripción:** El DOM como puente entre HTML y JavaScript. Concepto visual, no técnico.  
**Nivel:** Recomendado

### `10-inspector-navegador.md`
**Descripción:** Abrir el inspector, ver el código de cualquier web, modificarlo en tiempo real.  
**Nivel:** Esencial

### `11-experimentar-sin-miedo.md`
**Descripción:** Romper código a propósito, ver errores, arreglarlos. Entender que no pasa nada malo.  
**Nivel:** Esencial

### `12-donde-sigo-desde-aqui.md`
**Descripción:** Resumen de lo visto, conexión con el siguiente bloque (fundamentos), mapa mental.  
**Nivel:** Esencial

## Características de este bloque

### ✅ Lo que SÍ encontrarás aquí:
- Lenguaje sencillo y cercano
- Ejemplos visuales y concretos
- Resultados inmediatos en pantalla
- Invitación constante a experimentar
- Explicaciones de "para qué sirve" antes que "cómo funciona"

### ❌ Lo que NO encontrarás aquí:
- Sintaxis profunda de HTML/CSS/JS
- Conceptos de programación complejos
- Ejercicios largos o abstractos
- Terminología técnica sin explicar

## Orden de estudio

**Este bloque se recorre linealmente, de 01 a 12.**

No tiene sentido saltar módulos aquí. Cada uno construye sobre el anterior de forma muy gradual.

Si ya tienes experiencia programando, puedes saltar directamente a `/01-fundamentos`.
